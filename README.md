## ğŸ“‹ Frequently Asked Questions

### What server types does Enve support?
Enve supports Audiobookshelf, Plex, Jellyfin, Emby, SMB network shares, and local file imports.

### Does Enve sync my progress across devices?
Yes! Enve uses CloudKit to sync your listening progress, bookmarks, and library across all your Apple devices.

### Can I use Enve in my car?
Absolutely! Enve has full CarPlay integration with library browsing, chapter navigation, and playback controls.

### Does Enve work offline?
Yes! You can download audiobooks for offline listening and they'll be available in CarPlay too.

### How do I add audiobooks to Enve?
You can connect to supported servers (Audiobookshelf, Plex, Jellyfin, Emby, SMB) or import local files via the Files app, iTunes File Sharing, or Finder.

### Does Enve support multi-file audiobooks?
Yes! Enve handles audiobooks split into multiple MP3 files (one per chapter) and combines them seamlessly.

### Is my server login information secure?
Yes! Enve uses secure token-based authentication (JWT for Audiobookshelf) and all credentials are stored securely in your device's keychain.
