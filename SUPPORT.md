## ğŸ“‹ Troubleshooting Tips

### Server Connection Issues

**Audiobookshelf:**
- Ensure you're using the correct server URL format: `http://server:port` or `https://server:port`
- Check that your API token or credentials are correct
- Verify the server is accessible from your device's network

**Plex:**
- Make sure you're logged in with the correct Plex account
- Check that your Plex server has audiobook libraries set up
- Verify your Plex Pass status if using remote access

**Jellyfin/Emby:**
- Confirm the server URL includes the port (usually :8096 for Jellyfin)
- Ensure your user account has permission to access audiobook libraries
- Check server logs for connection attempts

**SMB:**
- Verify the share path format: `smb://server/share`
- Ensure you have the correct username and password
- Check that SMB file sharing is enabled on your server/NAS

### Sync Issues

If your progress isn't syncing between devices:
1. Check that you're signed into the same iCloud account on all devices
2. Verify iCloud Drive is enabled in Settings
3. Ensure you have sufficient iCloud storage
4. Try force-closing and reopening the app

### Playback Issues

If audiobooks won't play:
1. Check your internet connection (for streaming)
2. Verify the audiobook file isn't corrupted
3. Try redownloading the audiobook
4. Check iOS storage space for downloads
